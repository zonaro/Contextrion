Open source Windows context menu power tool

Paste. Customize. Transform. Right from Explorer.

Contextrion adds clipboard-to-file saving, folder icon customization, and a full suite of file and image manipulation tools directly into the Windows Explorer context menu.

Clipboard to File

Turn anything on your clipboard into a file.

Right-click in any folder background and choose Paste Into File. Text becomes `.txt`, images become `.png`/`.jpg`, copied files become `.zip`, audio becomes `.wav`, and everything else becomes `.bin`.

Security first

A native tool with a minimal surface area.

Contextrion is a self-contained WinForms application with no telemetry, no ads, and no unnecessary network behavior.

What it does

Three tools in one context menu entry.

01

Paste Into File

Save clipboard content as a file directly from the Explorer background. Supports text, images (PNG/JPEG format detection), file/folder ZIP archives, audio, and binary data with a preview dialog.

02

Folder Customization

Right-click a folder to apply custom icons from a built-in catalog (Windows 11, 10, 7/8 styles), import your own `.ico`/`.dll`/`.png`/`.jpg` files, or create composite icons with the Derived Icon Editor.

03

File and Image Tools

Apply transformations to selected files directly from the context menu: grayscale, resize, crop, circle crop, watermark, invert colors, clean metadata, optimize for web, minify JS/CSS, rename to URL, enumerate, combine images, and more.

04

Derived Icon Editor

Compose composite icons by layering images with opacity, scale, offset, rotation, and color controls. Available as a web demo and integrated into the app.

Full feature list

Everything Contextrion can do.

Clipboard to File

  • Text → `.txt`
  • Images (PNG/JPEG format) → `.png`/`.jpg`
  • Files/Folders → `.zip` (preserves structure)
  • Audio → `.wav` (with duration parsing)
  • Other binary → `.bin`
  • Preview dialog before saving

Folder Customization

  • Built-in icon catalog (Win 11, 10, 7/8)
  • Import `.ico`, `.dll`, `.png`, `.jpg`
  • Icon packs (DLL/ICO) support
  • Derived Icon Editor (layers, transforms)
  • Restore default folder icon
  • Create timestamp folder hierarchy (YYYY/MM/DD)

Image Tools

  • Grayscale (BT.601 weights)
  • Watermark (text or image, 50% alpha)
  • Crop from center
  • Crop to circle (square + ellipse)
  • Resize (maintains aspect ratio)
  • Invert colors
  • Clean EXIF/metadata
  • Optimize for web (resize + JPEG quality)
  • Combine images (vertical / horizontal)

File Tools

  • Rename to friendly URL (lowercase, accents removed)
  • Copy as Base64 Data URL
  • Copy file path to clipboard
  • Bulk rename with enumeration pattern
  • Copy file content to clipboard
  • Clean empty directories
  • Minify JS and CSS files

Quick start

Three steps to start using it.

  1. Download the latest release from GitHub.
  2. Run the app and click Install (requires admin via UAC).
  3. Right-click in a folder or on a file to access all tools.
